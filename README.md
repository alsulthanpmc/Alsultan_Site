# Al Sultan Premier Medical Center

A modern, high-performance single-page website for Al Sultan Premier Medical Center built with Astro 4, React, and Tailwind CSS.

## 🚀 Features

- **High Performance**: Static site generation with Astro for optimal loading speeds
- **Modern Stack**: Astro 4, React 18, TypeScript, Tailwind CSS
- **Smooth Animations**: CSS-based scroll animations and Framer Motion for complex interactions
- **Interactive Components**: React islands for modals, tabs, and sliders
- **Responsive Design**: Mobile-first approach with beautiful UI on all devices
- **SEO Optimized**: Schema.org markup, meta tags, and semantic HTML
- **Accessible**: WCAG 2.1 AA compliant with proper ARIA labels

## 📦 Tech Stack

- **Framework**: Astro 4.x
- **UI Library**: React 18
- **Styling**: Tailwind CSS 3.4+
- **Language**: TypeScript (Strict mode)
- **Animations**: Framer Motion
- **UI Components**: Radix UI (Dialog, Tabs)
- **Content**: Astro Content Collections

## 🛠️ Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## 📁 Project Structure

```
/
├── public/               # Static assets
├── src/
│   ├── components/       # Astro & React components
│   ├── content/          # Markdown content collections
│   │   ├── departments/  # Department details
│   │   └── services/     # Service descriptions
│   ├── data/            # JSON data files
│   ├── layouts/         # Page layouts
│   ├── pages/           # Page routes
│   ├── styles/          # Global styles
│   └── utils/           # Utility functions
├── astro.config.mjs     # Astro configuration
├── tailwind.config.mjs  # Tailwind configuration
└── tsconfig.json        # TypeScript configuration
```
